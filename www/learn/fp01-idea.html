Final Project Goal

Idea

  • I would like to use this website as a portfolio for future careers, and as a general "About Me" page.

Why are you choosing this idea?

  • I am choosing this idea because I think that having a portfolio page as a CS student kills two birds with one stone. 1. The student can build a basic web page. 2. Here are also some good things about said student.

Target audience

  • My target audience is potential employers and casual visitors seeking a bit more information on me.

What do you want your final project to accomplish?

  • I want my final project to be a minimalist webpage, almost bordering on a resume. I do not want it to be unnecessarily flashy. It should be clear, fast, and straight to the point.

Overarching Website Idea & Focus

  • I want to create a resume / portfolio page advertising myself. The website focus is to create an at-a-glance overview on me as a person. It should contain some information on me, as well as some sort of resume. The website should utilize CSS to a certain degree. The website should get its job done, and look decent. The website does not need to look amazing to me, but it should be functional. This is not a copout for the site to be ugly. What I mean is that, just because websites CAN have a million CSS customizations, does not mean they should. It is a simple website that fulfills a simple goal, it does not need to look like an art project.
